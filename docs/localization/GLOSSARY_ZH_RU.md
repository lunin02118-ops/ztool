# Глоссарий ZH -> RU

| ZH | RU | Примечание |
| --- | --- | --- |
| 图号 | Обозначение | Номер/обозначение изделия или детали |
| 零件名称 | Имя детали | Пользовательское свойство детали |
| 材料 / 材质 | Материал | BOM/property context |
| 类型 | Тип | BOM filter/property context |
| 版本 | Версия | BOM/property context |
| 表面处理 | Обработка | В UI может быть “Обработка поверхности” |
| 重量 | Масса | Не “вес”, чтобы совпадать с КД |
| 数量 / 统计数量 | Количество | Для BOM служебное агрегированное количество |
| 路径 | Путь | Путь файла |
| 缩略图 | Эскиз | Thumbnail/sketch в спецификации |
| 注册 | Регистрация | License registration |
| 激活 | Активация | Online activation |
| 转出 / 转移 | Перенос лицензии | Transfer-out flow |
| 连接 | Подключить | SolidWorks connection |
| 保存 | Сохранить | Save/export context |
