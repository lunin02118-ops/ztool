# UI screenshot checklist

Для release candidate нужны скриншоты или ручное подтверждение по точному
хешу артефакта.

## ZTool.exe

- Главное окно: вкладки, ribbon, таблица.
- Регистрация/активация: `FrmRg`, `FrmRverify`.
- Управление файлами: подключение к SolidWorks, таблица с позициями.
- Спецификация: меню 8 режимов, экспорт, настройки шаблона.
- Параметры: `FrmOptions`.
- Диалоги ошибок: неверный код, неверный пароль, сервер недоступен.
- Copy/paste списков: `FrmOutputlist`, `FrmPrintlist`, `FrmSetDrwlist`,
  `FrmSyncDrwName`.

## SolidWorks add-in

- Вкладка `ZTool — вспомогательные инструменты`.
- Все кнопки ленты на русском.
- Иконки видимы.
- In-DLL кнопки открывают PropertyManager без crash.
- `Управление файлами` запускает правильный `ZTool.exe`.

## Критерии

- Нет видимой Han-строки.
- Whitelist не является автоматическим visual PASS: если whitelisted строка
  видна пользователю, это blocker до перевода/исправления или отдельного
  release exception.
- Текст не обрезан критично; длинный текст читается полностью после изменения размера окна.
- Диалог можно растянуть мышью, если контента больше стартового размера.
- Кнопки доступны.
- Нет наложений.
- Форма открывается без исключений.

## Формы с обязательной проверкой читаемости

- `FrmSaveOption` — параметры сохранения.
- `Frmexportbom` — настройка схемы спецификации.
- `Frmmapping` — сопоставление заголовков столбцов.
- `FrmFilterrules` — пользовательские правила.
- `FrmSWUnit` — единицы измерения.
- `FrmOptions` — общие параметры и вкладки.
