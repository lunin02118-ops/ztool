# ZTool — чеклист пересъёмки скриншотов руководства (help_ru.chm)

Цель: заменить все **китайские** скриншоты в руководстве на **русские**.
Текст руководства уже переведён — меняем только картинки.

## Как сдавать (важно — иначе сборка не соберётся правильно)

1. Снимаешь скриншот русского состояния по инструкции из таблиц ниже.
2. **Сохраняешь файл с тем же именем и в той же папке**, что указано в колонке
   «Файл (путь в мануале)», просто перезаписывая китайский оригинал.
   - Формат — PNG в любом случае ок (даже если оригинал `.gif`/`.jpg`): я приведу
     к нужному формату при пересборке. Имя и расширение **сохраняй как в оригинале**.
   - Старайся выдержать **те же пропорции/кадрирование**, что у оригинала, чтобы
     вёрстка страницы не поехала (особенно для мелких «кнопочных» вырезок).
3. Всю папку с заменёнными картинками заархивируй и пришли мне — я пересоберу
   `help_ru.chm` и оформлю PR.

Папка-источник (распакованный мануал) с оригиналами для сверки — пришлю отдельным
архивом; структура такая:
```
activation.files\        property-names.files\     bom-template.files\
connect-sw.files\        save-to-sw.files\         dropdown.files\
material-library.files\  thumbnails.files\         installation.files\
```

## Общие требования к съёмке (читать один раз)

- **Всё снимать с ОДНОГО релизного билда** — русский ZTool 1.0 (x64) из релиз-пакета.
  В текущем мануале скриншоты сборные (2.5.2 / 2.7 / 2.7.2 / 3.2 / 3.5) — это
  выглядит неаккуратно; нужна единая версия. В заголовке окон должно быть
  `ZTool 1.0 (x64)`, а не `ZTool 3.x`.
- Тема Windows — светлая, масштаб экрана 100%, шрифт по умолчанию.
- Для данных в таблице используй тестовую сборку `TestModel\0614-A00.SLDASM`
  (та же, что в релиз-чеклисте), чтобы значения были консистентны во всех кадрах.
- Окна **разворачивать/кадрировать** так же, как в оригинале (см. архив-источник).
- Никаких реальных лицензионных ключей в кадре: на экранах активации поля
  «Регистрационный код»/«Машинный код» **заштрихуй/замажь** (в оригинале они тоже
  заблюрены).
- Демо-значения из китайского мануала переводим:
  - имена-примеры `张飞 / 关羽 / 刘备` → `Иванов / Петров / Сидоров` (любые RU);
  - тип `机加 / 钣金 / 外购` → `Механообработка / Листовой металл / Покупное`.

## Легенда «что это за приложение»
- **[ZT]** — окно/элемент самого ZTool (нужен русский ZTool). Большинство.
- **[SW]** — родной диалог SolidWorks (нужен SolidWorks с русским — или английским —
  интерфейсом; уточни, какой язык SW у целевой аудитории).
- **[XLS]** — Microsoft Excel (нужен русский Excel + файл шаблона BOM).
- **[icon]** — нейтральная иконка-глиф без текста; **можно оставить как есть**
  (перечислено отдельно, сними только если хочешь полного единообразия).
