## 2. Сборка релиза

```powershell
cd <repo>\client-core
# обычная пересборка (ref/*.dll уже в репо)
.\build.ps1
# артефакт: client-core\out\SWTools.exe + out\SWTools.manifest.json
```

**Критерии PASS сборки:**
- В логе локализации присутствуют строки:
  - `FrmRverify: redesigned (... 'Демо'/'Регистрация' buttons ...)`
  - `about-box edits=1`, `verify edits=1`, `frmrg edits>0`, `max-qr edits>0`
- `VERIFY: PASS`
- `dangling typerefs = 0`
- `DONE -> ...\out\SWTools.exe`

Затем соберите установщик (NSIS) и пересчитайте хеши штатным релизным скриптом проекта (см. `client-core/build-inputs.json` / release-скрипты). **FAIL**, если хеши в манифесте не совпадают с артефактом.
