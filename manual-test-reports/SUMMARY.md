# Manual test summary (sanitized)

Подробные отчёты (скриншоты, локальные пути, серверные детали) намеренно
**не** хранятся в репозитории. Ниже — краткая сводка проверок по этой ветке.

## Лицензирование (серверная часть)

Перевыпущенный клиент `ZTool.exe` проверен против боевого license-сервера —
все 4 режима протокола проходят:

| Режим | Запрос | Результат |
| --- | --- | --- |
| Регистрация | `apply_register` (128) | `13` + license blob |
| Подтверждение | `register_confirm` (131) | `12` (IsReg2 OK) |
| Перенос (отвязка) | `apply_remove` (129) | `11` + transfer blob |
| Подтверждение переноса | `remove_confirm` (132) | `7` (слот освобождён) |

TCP-сервер переведён на единую БД с админкой: после активации в админке
видны привязка машины и счётчики; после переноса — очищаются.
